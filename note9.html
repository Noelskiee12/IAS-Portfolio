INTRODUCTION TO EVENT LOGS

What Are Event Logs?

— Records of significant system actions or events.

— Stored automatically by software systems.

— Serve as a digital history of system activity.

Importance of Logs in Security

— Monitor workstation and endpoint activity.

— Track attacks and unauthorized access.

— Help in incident investigation & audits.

— Enable event correlation across systems.

Key Parts of a Log Entry

— Timestamp: when the event happened.

— Source: system or application that created the log.

— Event Type: login, file access, error, etc.

— Details: user, IP address, device, outcome.

Power of Log Correlation

— Single logs have limited value.

— Combined logs show:

What happened, when, where, who caused it, and result/outcome.

— Used to reconstruct attack chains (e.g., brute-force login attacks).

Common Log Types

— Application Logs: app errors, warnings.

— Audit Logs: compliance-related activity.

— Security Logs: logins, permission changes.

— System Logs: OS, hardware, kernel events.

— Network Logs: traffic & connections.

— Web Server Logs: URL requests & responses.

Log Formats

— Structured: machine-readable format.

— Semi-structured: mixed format.

— Unstructured: free text, hard to analyze.

Log Collection & Centralization

— Logs gathered from different systems.

— Time sync (NTP) is critical for accuracy.

— Stored in central systems (SIEM).

— Examples: Splunk, Elastic, IBM QRadar, Azure Sentinel.

Windows Event Logs

— Stored in .evt / .evtx.

— Located at: C:\Windows\System32\winevt\Logs

— Converted to XML for viewing.

— Standard Windows Log Types:

System: services, hardware (e.g., Event ID 7045)

Security: logins, attacks

Application: app crashes

— Log Entry Elements: Level, Time, Source, Event ID, Task Category, Details (XML)

Windows Log Tools

— Event Viewer: graphical viewer.

— Wevtutil.exe: command-line tool.

— Get-WinEvent: PowerShell log analysis.

Key Takeaways

— Logs are the primary forensic evidence of cyber activity.

— SIEM systems analyze and correlate logs.

— Windows Event Logs act as a system flight recorder.

— Essential for cybersecurity investigations.
