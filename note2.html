ENCRYPTION FUNDAMENTALS

PRINCIPLES OF CRYPTOGRAPHY

Encryption

Hides data so only authorized people can read it.

Authentication

Confirms who someone is.

Integrity

Ensures data hasn’t been changed.

Non-reputation

Proves that someone actually sent a message and can’t deny it later.

CIPHERS

Symmetric Encryption

Uses one key for both encrypting and decrypting data.

Asymmetric Encryption

Uses two keys: a public key to encrypt and a private key to decrypt.

ASYMMETRIC CRYPTOGRAPHIC ALGORITHMS

RSA

Uses large prime numbers for encryption and digital signatures.

DSA

Used mainly for digital signatures, not encryption.

ECDSA

A faster, stronger version of DSA that uses elliptic curve cryptography.

SYMMETRIC CRYPTOGRAPHIC ALGORITHMS

DES

Older encryption method; now considered weak.

IDEA

A secure and fast encryption algorithm used in early email security.

AES

The most common and strong encryption standard today.

RC

A family of ciphers used for fast data encryption.

Transport Encryption

Protects data while it’s moving (example: HTTPS).

End-To-End Encryption

Protects data from sender to receiver.

Purpose of X.509 Certificates

Digital IDs ensuring you're connecting to a real, trusted site.

CLOUD AND VIRTUALIZATION FUNDAMENTALS

Hypervisors: The Foundation of Virtualization

Software that allows multiple virtual machines to run on one system.

Type 1 (Bare-Metal)

Runs directly on hardware. Faster, secure, used in data centers.

Type 2 (Hosted)

Runs on top of an OS. Easier to use but less efficient.

Variety of Cloud Storage

Saves data on remote servers via the internet.

Cloud Storage Applications

Used to upload, manage and share cloud files.

File Synchronization

Keeps files updated across multiple devices.

Content Delivery Networks (CDNs)

Delivers content quickly by caching data near users.

Cloud Deployment Models

Public Cloud

Shared, cost-effective, managed by providers.

Private Cloud

Exclusive to one organization, more secure.

Community Cloud

Shared by organizations with similar needs.

Hybrid Cloud

Mix of public and private cloud systems.

Cloud Access Security Broker (CASB)

Protects cloud usage by monitoring and enforcing security policies.

Functions

  • Visibility
  • Threat detection
  • Policy enforcement
  • Data protection

Single Sign-On (SSO)

One login for multiple cloud apps.

Access Controls

Controls who can access cloud resources.

BURP SUITE – DIGITAL NOTES

What is Burp Suite?

A full web security testing toolkit used by pros and bug bounty hunters.

Core Tools in Burp Suite

Proxy

Intercepts and modifies web traffic.

Repeater

Manually resend and modify HTTP requests.

Intruder

Automates attacks (SQLi, XSS, brute force).

Scanner

Auto-finds vulnerabilities (Pro only).

Sequencer

Analyzes session token randomness.

Installing Burp Suite

Community (free) and Professional versions available. Requires Java.
